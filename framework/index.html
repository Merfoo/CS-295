HTML Boilerplate Description

HTML Boilerplate is a minimalistic framework meant to make website development simple and fast. It only includes the bare minimum to have a professional website up in no time and allows the developer control and freedom to do whatever they want. Its the perfect foundation for use with other popular frameworks or by itself.

More information can be found at their homepage here

HTML Boilerplate HTML

Includes only the basic index.html for the entry point, and 404.html for the error page. The index page comes with all the css and javascript files linked, meta tags for website information, and google-analytics built in.

More information can be found here

HTML Boilerplate CSS

Has two css files, main.css and normalize.css. The main css file has styling for links, some helper classes for elements, and media queries for printing. The normalize css file has styling to make the look be consistent among different browsers and sizes.

More information can be found here

HTML Boilerplate Scripts

Comes with two basic javascript libraries, jQuery and Mordernizer. The libraries are placed in a vendor folder thats inside a js folder to allow the developer a place for all their js scripts. jQuery is included to give the developer tons of utility, and Modernizer is included to make detection of feature support from browsers easy.

Comparison of Frameworks
  Markup CSS grids CSS Compilers Javascript Libraries Server-side Scripts Data Handling Browser Support Notes
Boilerplate Standard HTML markup provided. CSS grids not included. CSS compilers not included. Comes with jQuery via CDN and local one for fall back and modernizer. Server-side scripts not included. Does not include any tools for external data handling. Browser support is provided via: Browser upgrade not, htaccess, touch/favorite MS touch tiles, normalize stylesheet Simple frameworks meant to get a basic website up and running quickly.
Bootstrap Includes standard HTML markup and many preconfigured components. CSS grids use float properties to mimic pure CSS grid style. Includes precompiled LESS/SASS, can use Grunt to build LESS/SASS. Includes plenty of JS libraries to modify HTML elements behaviors. Server-side scripts not included. Does not include any tools for external data handling. Supports latest version of desktops browsers Chrome, Firefox, Internet Explorer, Opera, and Safari. For mobile the latest versions of Chrome, Firefox, and Safari are supported. More complex framework for and includes more libraries for a more interactive website.
Angular.js Includes standard HTML markup and allows the creation of reusable HTML components written in JS. CSS grids not included CSS compilers not included. AngularJS is a JS oriented framework, has tons of modular JS libraries you can use. Server-side scripts not included. Has limited functionality in JS for external data handling. Supports latest versions of Chrome, Firefox, Edge, Internet Explorer, Safari, iOS, Android, and Internet Explorer mobile Framework intended for developers that want to create an interactive website/web app in typescript or JS.
Ruby on Rails Includes standard HTML markup and has support for template files to make modular components. CSS grids not included. Includes support for SASS through a ruby gem thats installed by default. No JS included by default, supports JS libraries though. Framework is written in and requires Ruby on the server for development. External data handling is one of main pros of Ruby on Rails, has support for connecting to many popular databases. Ruby on Rails is a server side application, browser compatibility depends on the front end framework being used. Ruby on Rails intended for developers that want to create scalable applications with their choice of front end. Has great support and documentation for websites/web app that use a database.
Comparison of CMS applications
  Server software required Script language and version required Database name and version required Security commitments Developer features User features Licensing/Pricing Notes
Wordpress Requires PHP, MySQL or MariaDB, HTTPS support, and Apache or Nginx as the server. Requires at minimum PHP version 7. Requires MySQL version 5.6 or greater or MariaDB version 10.0 or greater. Extensive summary of Wordpress's commitment to security can be found here. Developer features includes an expansive plugin ecosystem developers can use to enable different type of features and a wide variety of themes. User features includes a wide plugin ecosystem, user management, multilingual support, and easy installation/upgrade process. Wordpress software is licensed under GPLv2. Pricing plans include $0 for basic, $2.99 monthly for personal, $8.25 monthly for premium, and $24.92 monthly for business. Wordpress is one of the most common and easiest CMS out there to use, would recommend for beginners.
Drupal Requires a web server, PHP, and database. Requires at minimum PHP version 5.2.5. Requires MySQL 5.0.15 or MariaDB 5.1.44 or Percona Server 5.1.70 with PDO. Includes documentation here detailing how to write secure code. Extensive list of useful tutorials for developing with Drupal 7 listed here. Has a "Modules" marketplace where you can install various modules to add/enhance features in Drupal 7. Drupal is licensed under GPLv2 with it being free to download and use. Drupal 7 has a steeper learning curve than Wordpress and isn't as popular.
Squarespace Server and software provided by Squarespace. All tools necessary are provided by Squarespace. Database provided by Squarespace. Offers SSL certificates for custom domains for extra security. All the software is provided by Squarespace and has excellent documentation online, also provides a graphical user interface for editing your website. Tons of professional looking themes provided by Squarespace that are also really customizable. Offers two groups for plans, one for websites and another for stores. Website plans are $12 monthly for personal and $18 monthly for business. For stores its $26 monthly for basic and $40 monthly for advanced. Squarespace looks to be intended for developers that want an emphasis on having a good looking and appealing website.
