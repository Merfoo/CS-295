Description

I want to make a cooking tutorial website because I want a place where I can search for tutorials on how to make foods. I want the website organized such that its super easer to search for cooking tutorials based on type of food and duration of cooking time. This website will be accessible through searching in google "how to make..." or "how to cook...". The website will be for people that want a place where they can search for cooking tutorials. It will include foods of any time and will be categorized under when you would eat the meal, how long it takes to cook, and other potential categories like ingredients. The website will be run using Rails or NodeJS and VueJS.

List of Goals

Upon entering the site, visitors will...

  1. Learn how to cook food.
  2. Donate money to users that have posted cooking tutorials.
  3. Comment on cooking tutorials.
  4. Share cooking tutorials on various social media sites like Facebook and Twitter.

User Flow

The flow would basically consist of getting to the cooking tutorial website though google, seeing someone share the page on a social media website, typing the website directly into the url bar, or hearing about the website from a friend. Once the user is at a site they'll search for a cooking tutorial based on the type of meal they're looking for or on cooking duration.

Audience Demographics

The audience demographics would be targeted towards college students, middle class people, and families.

College students According to the National Center for Education Statistics, 20.5 million students are expected to attend colleges in America.
Middle class people According to PewResearchCenter roughly 51% of adults lived in middle-income households.
Families According to Statista the average family is roughly just over 2 kids.

Ruby on Rails is a server side application, browser compatibility depends on the front end framework being used. Ruby on Rails intended for developers that want to create scalable applications with their choice of front end. Has great support and documentation for websites/web app that use a database.

HTML Boilerplate Description

HTML Boilerplate is a minimalistic framework meant to make website development simple and fast. It only includes the bare minimum to have a professional website up in no time and allows the developer control and freedom to do whatever they want. Its the perfect foundation for use with other popular frameworks or by itself.

More information can be found at their homepage here

HTML Boilerplate HTML

Includes only the basic index.html for the entry point, and 404.html for the error page. The index page comes with all the css and javascript files linked, meta tags for website information, and google-analytics built in.

More information can be found here

HTML Boilerplate CSS

Has two css files, main.css and normalize.css. The main css file has styling for links, some helper classes for elements, and media queries for printing. The normalize css file has styling to make the look be consistent among different browsers and sizes.

More information can be found here

HTML Boilerplate Scripts

Comes with two basic javascript libraries, jQuery and Mordernizer. The libraries are placed in a vendor folder thats inside a js folder to allow the developer a place for all their js scripts. jQuery is included to give the developer tons of utility, and Modernizer is included to make detection of feature support from browsers easy.

More information can be found here
